# ➗ Expressions & Equations

**Grade:** 6-8 · **Standard:** CCSS 6.EE · **Read time:** 5 min

## 🌟 Big Idea

Algebra is the language of patterns. When you write an expression or solve an equation, you're really saying: "I know the rules, and I can predict what happens next." That's exactly what sound engineers do every day — they use math to predict how sound will behave in a room.

> 💡 **Did you know?** The word "algebra" comes from the Arabic *al-jabr*, meaning "reunion of broken parts" — like putting a sound wave back together after it bounces off a wall!

## 🧪 Core Concepts

### Expressions vs. Equations

| Concept | Definition | Example | Studio Analogy |
|---------|-----------|---------|----------------|
| **Variable** | A symbol for an unknown value | `x`, `y` | The gain knob — you don't know the exact setting yet |
| **Expression** | A combination of numbers, variables, and operations | `3x + 5` | A signal chain: mic → preamp → EQ |
| **Equation** | Two expressions set equal | `3x + 5 = 20` | Balancing levels: left channel = right channel |
| **Solution** | The value that makes the equation true | `x = 5` | The perfect gain setting for no distortion |

### Order of Operations (PEMDAS)

Just like a mixing board processes signal in a specific order (preamp → EQ → compression → reverb), math follows a strict order:

```
P - Parentheses    ( )     — like grouping tracks in a DAW
E - Exponents      x²      — like doubling the frequency
M/D - Multiply/Divide      — like splitting a signal
A/S - Add/Subtract         — like blending two tracks
```

### 🎵 Lab Connection: The Studio Equation

At **Mohikontok Sound Lab**, every recording session is an equation. The engineer asks: "If I set the gain to 30 dB (`g = 30`) and the room adds 5 dB of natural reverb (`r = 5`), what's the total signal level?"

**Equation:** `g + r = L` → `30 + 5 = 35 dB`

But if the artist moves closer to the mic, the proximity effect adds 3 dB: `30 + 5 + 3 = 38 dB`. Every variable changes the result — just like in algebra.

## 🎯 Learning Activity

Solve these studio-themed equations:

| Equation | What It Represents | Solution |
|----------|-------------------|----------|
| `m + 12 = 48` | Mic level + 12 dB boost = 48 dB total. What's the mic level? | `m = ___` |
| `3c = 72` | 3 cables cost $72 total. What's one cable? | `c = ___` |
| `2x + 10 = 50` | 2 tracks + 10 dB headroom = 50 dB max. What's each track's level? | `x = ___` |

**Challenge:** Write your own equation based on a real studio scenario!

## 🎥 Audio-Visual Learning

**Activity: The Volume Equation**

1. Pick a song you know well
2. Set your phone volume to a comfortable level — call this `V`
3. Turn it up by 3 notches — that's `V + 3`
4. Now turn it down by 5 notches — that's `V - 5`
5. Write an equation: if `V = 15` (out of 30), what is `V - 5`?

**At the lab:** We use equations to calculate everything — from how much acoustic foam we need for a room (`A = l × w × 0.2`) to how long reverb should last (`RT60 = 0.161 × V / Sα`). Algebra isn't just homework — it's how we build better sound.
