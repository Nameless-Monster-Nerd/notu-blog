🧠 Big Idea: Identifying 2D and 3D shapes in the real world 📊 Visual Guide: Shape reference grid: 🔺△ ⬜□ ⭕◯ 🔷⬡ — 2D on top row, 3D (📦⚽🥫🧊) on bottom row
📋 Follow These Steps:
🔍
Step 1 Learn the Shapes

2D shapes are flat: △ triangle, □ square, ○ circle, ⬡ hexagon. 3D shapes have depth: 📦 cube, ⚽ sphere, 🥫 cylinder, 🧊 pyramid.

👁️
Step 2 Shape Bingo

Your bingo card has: circle, square, triangle, rectangle, hexagon. Walk around. Find each shape in the real world. Mark it off!

✏️
Step 3 Design a Building

Draw a building using ONLY these shapes: rectangles (walls), triangles (roof), circles (clock), cylinders (columns). Label each shape.

✂️
Step 4 Cut & Match 3D

Cut out the 3D shape names. Match each to a real-world object.

🎨
Step 5 Shape Color Code

Color all TRIANGLES red. Color all CIRCLES blue. Color all SQUARES yellow. Color all RECTANGLES green.
