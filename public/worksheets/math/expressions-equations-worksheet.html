🧠 Big Idea: Writing and evaluating algebraic expressions with variables 📊 Visual Guide: Visual equation: 🧁 = cupcake, 🧁 + 3 = ? shows variable (🧁) as unknown number
📋 Follow These Steps:
🔍
Step 1 Mystery Numbers

A variable is a letter that stands for an UNKNOWN number. If x = 5, then x + 3 = 8. If y = 10, then y - 4 = 6.

🍕
Step 2 Pizza Math

Pizza costs $12. Each topping costs $2. Expression: 12 + 2t. If t=3 (3 toppings): 12 + 2(3) = $18. If t=1: ___. If t=5: ___.

🔢
Step 3 Solve the Mystery

If n + 5 = 12, what is n? n = ___. If 3 × y = 21, y = ___. If x - 4 = 8, x = ___.

✂️
Step 4 Cut & Match Expressions

Cut out the expressions. Match each to its value when n = 4.

✏️
Step 5 Write Your Own

A burger costs $8. Each extra patty costs $2. Write the expression: ___. How much for 2 extra patties? $___.

Teacher Guide — UDL Adaptations
Use a cup with hidden coins as the variable. Add more coins, count total, then find how many were hidden. For sensory: balance scales.
