Expressions — Math Sentences Without Equal Signs

📚 Grade 6-7 📐 Math ⭐ UDL Designed 🏫 District 75
Common Core State Standards — Mathematics
EE: Expressions and Equations — Write and solve algebraic equations

🌟 Big Idea: Math Can Have Mystery Numbers

An **expression** is a math phrase that combines numbers and operations: `3 + 5`, `x - 2`, `4 × y`.

A **variable** is a letter that stands for an unknown number: `x`, `y`, `n`. It's like a mystery number waiting to be solved!

🧮 **Examples:**
- If x = 5, then x + 3 = 8
- If y = 10, then y - 4 = 6
- If n = 2, then 3 × n = 6

**Why do we need variables?**

They let us write rules that work for ANY number. Instead of saying 'add 3 to whatever number you have,' we write `n + 3`.
🍕 **Real-world:**
Pizza costs $12, and each topping costs $2. The expression is: 12 + 2t (where t = number of toppings).
If you want 3 toppings: 12 + 2(3) = 12 + 6 = $18!

**Variables are math's shortcut for 'it depends'!**

✏️ Interactive Activities (2)

1

🍕 Pizza Math

Write an expression for: A burger costs $8. Each extra patty costs $2. Write the expression. How much for 2 extra patties? 3 extra?

2

🔢 Variable Explorer

If n + 5 = 12, what is n? If 3 × y = 21, what is y? If x - 4 = 8, what is x? Draw pictures to help you solve.

👩‍🏫 Teacher Guide — UDL Adaptations

Engagement: Let students choose their preferred activity. Use the interactive boxes for typed responses, or print and complete by hand. Offer individual or pair work.

Representation: Read all text aloud. Use the ✖️ icon as a visual cue. Color-code and simplify instructions as needed.

Action/Expression: Students can type in the interactive boxes, print and write, draw on paper, or verbally answer. Use the contenteditable boxes for typed responses.

Use physical objects (coins, blocks) to represent variables. A cup with an unknown number of coins inside = the variable. For sensory: use balance scales.

🧠
Sensory Connection: 🧱 Use blocks: a cup hides some blocks (the variable). Add 3 more. Count the total. Find how many were hidden! 🪙 Use coins in a closed hand as variables.
